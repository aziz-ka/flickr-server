# [Flickr Server](https://flickr-server-ybxcxnqonw.now.sh)


## Stack

- Http Server
- Socket.io


#### Quick Start

```shell
$ git clone git@gitlab.com:flickr-amex/flickr-server.git
$ cd flickr-server
$ yarn
$ yarn start
```


#### Yarn/NPM Commands

|Script|Description|
|---|---|
|`yarn start`|Start the app on `localhost:5000`|
